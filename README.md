The PP-LCGA and bias-adjusted three step regression models were conducted in Mplus version 8.10.
Mixed effects models and the ROBPCA were conducted in R using the lme4 and ropsca packages respectively.
The code provided here has been condensed so that only the main analyses are shown for ease of interpretation. Additional preliminary steps conducted have not been included in the condensed analysis scripts. 
For those interested, a full analysis code including preliminary tests can be made available upon request.
